## O que é a seção de Serviços?

A seção de Serviços apresenta os principais produtos ou serviços oferecidos pela empresa. Geralmente são exibidos em cards com título, descrição e link para mais detalhes.

## Para que serve?

- Mostrar claramente o que a empresa oferece
- Guiar visitantes para páginas específicas
- Demonstrar especialização em diferentes áreas
- Facilitar a navegação no site

## Como editar os Serviços

1. **Acesse o Sanity Studio** e clique em "Home Page" > "Services"
2. **Para cada serviço**: Clique para expandir e editar
3. **Title**: Nome do serviço
4. **Description**: Descrição detalhada
5. **Link**: URL da página de destino
6. **CTA Text**: Texto do botão (opcional)
7. **Icon**: Ícone representativo (opcional)
8. **Publicar**: Clique em "Publish" para salvar

<Callout type="info" title="Dica">
Mantenha as descrições concisas mas informativas. Foque nos benefícios para o cliente.
</Callout>

## Campos disponíveis

### Title (Título)
Nome claro e descritivo do serviço.

**Exemplos:**
- "Viagens Corporativas"
- "Viagens de Lazer"
- "Eventos Empresariais"

### Description (Descrição)
Texto explicativo sobre o que o serviço oferece. Deve ser persuasivo e focar nos benefícios.

**Dicas para boa descrição:**
- Comece com o benefício principal
- Mantenha entre 2-4 frases
- Use linguagem clara e profissional
- Inclua palavras-chave relevantes

### Link (Link)
URL da página de destino onde o visitante pode saber mais sobre o serviço.

**Exemplos:**
- `/corporate` (página de viagens corporativas)
- `/lazer` (página de viagens de lazer)
- `/eventos` (página de eventos)

### CTA Text (Texto do Botão)
Texto que aparece no botão de chamada para ação.

**Exemplos:**
- "Saiba Mais"
- "Ver Detalhes"
- "Solicite Orçamento"

## Estrutura recomendada

Para uma seção de serviços eficaz:

1. **Limite a 3-4 serviços**: Não sobrecarregue o visitante
2. **Ordem lógica**: Do mais popular para o menos popular
3. **Ícones consistentes**: Use ícones relacionados ao serviço
4. **Links funcionais**: Certifique-se de que levam a páginas relevantes

## Dicas de otimização

✅ **Títulos claros**: Devem ser autoexplicativos
✅ **Descrições persuasivas**: Mostrem benefícios, não apenas recursos
✅ **Links funcionais**: Teste todos os links antes de publicar
✅ **CTA atraente**: Use verbos de ação

<Callout type="warning" title="Importante">
Cada serviço deve ter uma página de destino dedicada com informações completas.
</Callout>

## Erros comuns

- Descrições muito longas ou confusas
- Links quebrados ou incorretos
- Títulos genéricos demais
- Falta de hierarquia visual

### Icon (Ícone)
Nome do ícone a ser exibido (se disponível no tema).

<Callout type="info">
Os ícones são configurados automaticamente com base no tipo de serviço. Você geralmente não precisa alterar este campo.
</Callout>

## Dicas

✅ **Descrições claras** - Explique benefícios, não apenas características
✅ **Call-to-action** - Termine com uma frase que incentive ação
✅ **Consistência** - Mantenha tamanho similar entre as 3 descrições

## Adicionando Novo Serviço

Embora o layout seja otimizado para 3 serviços, você pode adicionar mais:

1. Role até o final da lista em Services
2. Clique em **"Add item"**
3. Preencha os campos
4. Publique

<Callout type="warning">
Mais de 4 serviços pode prejudicar o layout visual. Considere criar uma página separada para serviços adicionais.
</Callout>

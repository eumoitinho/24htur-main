# Serviços

Esta seção exibe os principais serviços em cards (viagens corporativas, lazer, eventos).

## Como editar

1. No Sanity Studio, acesse **Home Page** > **Services**.
2. Clique em um item para editar ou **Add item** para novo.
3. Preencha os campos e clique em **Publish**.

## Campos Principais

### Title (Título)
Nome do serviço (ex: "Viagens Corporativas").

### Description (Descrição)
*   Explique o que é oferecido e os benefícios.
*   Mantenha entre 2-4 frases.
*   Seja persuasivo.

### Link
URL de destino (ex: `/corporate`, `/lazer`).

### CTA Text
Texto do botão (ex: "Saiba Mais").

### Icon (Ícone)
Geralmente configurado automaticamente.

<Callout type="warning" title="Limites">
  Limite a 3-4 serviços para não poluir o layout. Certifique-se que os links levam a páginas existentes.
</Callout>

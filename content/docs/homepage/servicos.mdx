# Seção de Serviços

Apresenta os 3 principais serviços da 24H Turismo: Corporativo, Lazer e Eventos. Cada serviço tem título, descrição e link.

## Como Editar

1. Acesse `/studio` → **Home Page**
2. Localize a seção **"Services"**
3. Expanda para ver os 3 serviços
4. Clique em cada serviço para editar

## Campos de Cada Serviço

### Title (Título)
Nome do serviço.

**Exemplos atuais:**
- Viagens Corporativas
- Viagens de Lazer
- Eventos Empresariais

### Description (Descrição)
Breve explicação do que o serviço oferece (2-3 frases).

**Exemplo:**
```
Soluções completas para viagens de negócios, reuniões e
convenções. Otimize custos e garanta conforto para sua equipe.
```

### Link
Link para página com mais detalhes.

**Exemplos:**
- `/corporate` - Viagens Corporativas
- `/lazer` - Viagens de Lazer
- `/eventos` - Eventos

### Icon (Ícone)
Nome do ícone a ser exibido (se disponível no tema).

<Callout type="info">
Os ícones são configurados automaticamente com base no tipo de serviço. Você geralmente não precisa alterar este campo.
</Callout>

## Dicas

✅ **Descrições claras** - Explique benefícios, não apenas características
✅ **Call-to-action** - Termine com uma frase que incentive ação
✅ **Consistência** - Mantenha tamanho similar entre as 3 descrições

## Adicionando Novo Serviço

Embora o layout seja otimizado para 3 serviços, você pode adicionar mais:

1. Role até o final da lista em Services
2. Clique em **"Add item"**
3. Preencha os campos
4. Publique

<Callout type="warning">
Mais de 4 serviços pode prejudicar o layout visual. Considere criar uma página separada para serviços adicionais.
</Callout>

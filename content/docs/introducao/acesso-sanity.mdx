# Como Acessar o Sanity Studio

O Sanity Studio é o painel administrativo onde você gerencia todo o conteúdo do site. Ele está integrado diretamente no site da 24H Turismo.

<VideoPlaceholder
  title="Acessando o Sanity Studio"
  description="Demonstração de como fazer login no painel (2-3 min)"
/>

## Endereço do Painel

O painel administrativo está localizado em:

```
https://seusite.com/studio
```

<Callout type="info">
Substitua `seusite.com` pelo domínio real do seu site 24H Turismo. O caminho `/studio` sempre levará ao painel administrativo.
</Callout>

## Fazendo Login

<StepByStep>
1. Abra seu navegador preferido (Chrome, Firefox, Safari ou Edge)

2. Digite o endereço do painel na barra de URL

3. Você será direcionado para a tela de login do Sanity

4. Escolha o método de autenticação:
   - **Google**: Faça login com sua conta Google
   - **GitHub**: Faça login com sua conta GitHub
   - **E-mail**: Use o e-mail cadastrado

5. Após autenticação bem-sucedida, você será redirecionado para o painel

6. Aguarde alguns segundos enquanto o Studio carrega
</StepByStep>

<Callout type="warning" title="Primeira vez?">
Se esta é sua primeira vez acessando, você precisará de permissões. Entre em contato com o administrador do sistema para liberar seu acesso através do e-mail que você deseja usar.
</Callout>

## Problemas ao Acessar?

### Não consigo fazer login

- Verifique se você está usando o e-mail correto (o mesmo que foi cadastrado)
- Limpe o cache do navegador e tente novamente
- Tente usar modo anônimo/privado do navegador
- Entre em contato com o suporte técnico

### A página não carrega

- Verifique sua conexão com a internet
- Certifique-se de que está usando a URL correta
- Tente recarregar a página (F5 ou Ctrl+R)
- Teste em outro navegador

### Erro de permissão

- Você pode ter feito login com o e-mail errado
- Suas permissões podem não ter sido configuradas ainda
- Aguarde até 5 minutos após a configuração inicial
- Entre em contato com o administrador

<Callout type="success" title="Acesso conseguido!">
Assim que você vir a interface do Sanity Studio com o menu lateral e o logo da 24H Turismo, você está pronto para começar a editar!
</Callout>

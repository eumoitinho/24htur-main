# Acessando o Sanity Studio

O Sanity Studio é o painel onde você gerencia o conteúdo do site.

## Endereço

Acesse pelo navegador:
`https://seusite.com/studio`

<Callout type="info">
  Substitua `seusite.com` pelo seu domínio.
</Callout>

## Login

<StepByStep>
1. Acesse a URL do painel.
2. Escolha o método de login (Google, GitHub ou E-mail).
3. Aguarde o carregamento da interface.
</StepByStep>

<Callout type="warning" title="Permissões">
  Se for seu primeiro acesso, peça ao administrador para liberar seu e-mail.
</Callout>

## Problemas Comuns

*   **Login falhou:** Verifique se o e-mail é o mesmo cadastrado.
*   **Página não carrega:** Tente recarregar (F5) ou limpar o cache.
*   **Erro de permissão:** Aguarde alguns minutos após o cadastro ou contate o admin.

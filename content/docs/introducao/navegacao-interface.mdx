# Interface do Sanity Studio

Conheça as principais áreas do painel administrativo.

## 1. Menu Lateral (Esquerda)
Onde você navega entre as páginas e configurações do site.
*   **Home Page:** Página inicial.
*   **Sobre/Equipe/Eventos:** Páginas internas.
*   **Site Settings:** Configurações globais (menu, rodapé).

## 2. Editor de Conteúdo (Centro)
Onde você edita os campos.
*   **Texto:** Títulos e descrições.
*   **Imagens:** Upload e seleção de fotos.
*   **Listas:** Adicionar/remover itens (serviços, depoimentos).

## 3. Barra Superior
Ações principais.
*   **Publish (Verde):** Salva as alterações no site. Sempre clique aqui!
*   **Avatar:** Configurações da sua conta.

<Callout type="warning" title="Importante">
  As mudanças só aparecem no site após clicar em **Publish**. O salvamento automático cria apenas um rascunho.
</Callout>

## Dicas

*   **Preview:** Use para visualizar antes de publicar (quando disponível).
*   **Desfazer:** Ctrl+Z funciona para desfazer erros.
*   **Obrigatórios:** Campos com asterisco (*) não podem ficar vazios.

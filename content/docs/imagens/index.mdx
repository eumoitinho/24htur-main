# Gerenciamento de Imagens

Aprenda a usar imagens no seu site.

## Guias

*   **[Como Fazer Upload](/docs/imagens/upload):** Passo a passo.
*   **[Tamanhos Recomendados](/docs/imagens/tamanhos):** Dimensões ideais.
*   **[Otimização](/docs/imagens/otimizacao):** Como reduzir tamanho.

## Formatos Aceitos

*   **JPG:** Para fotos.
*   **PNG:** Para logos e ícones (fundo transparente).
*   **WebP:** Recomendado para tudo (menor tamanho).

## Limite de Tamanho

*   **Máximo:** 10MB.
*   **Ideal:** Entre 100KB e 300KB.

<Callout type="warning" title="Velocidade">
  Imagens pesadas deixam o site lento. Use ferramentas como TinyPNG para otimizar antes de enviar.
</Callout>

## Boas Práticas

*   ✅ Use nomes descritivos no arquivo (ex: `equipe-2024.jpg`).
*   ✅ Preencha o Texto Alternativo (Alt Text).
*   ✅ Verifique direitos autorais das imagens.

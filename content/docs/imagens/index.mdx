## Formatos Aceitos

O Sanity aceita os seguintes formatos de imagem:

| Formato | Uso Recomendado | Observações |
|---------|-----------------|-------------|
| **JPG** | Fotos, backgrounds | Melhor para imagens com muitas cores |
| **PNG** | Logos, ícones | Suporta transparência |
| **WebP** | Qualquer uso | Melhor compressão (recomendado) |
| **SVG** | Logos, ícones | Vetorial, não perde qualidade |

<Callout type="success" title="WebP é o melhor formato">
Se possível, use WebP para todas as imagens. Oferece melhor qualidade com menor tamanho de arquivo.
</Callout>

## Limite de Tamanho

- **Máximo por arquivo**: 10MB
- **Recomendado**: Menos de 500KB por imagem
- **Ideal**: Entre 100-300KB

<Callout type="warning">
Imagens muito grandes deixam o site lento. Sempre otimize antes de fazer upload!
</Callout>

## Biblioteca de Mídia

O Sanity mantém uma biblioteca com todas as imagens enviadas. Você pode:

✅ Reutilizar a mesma imagem em vários lugares
✅ Ver onde cada imagem está sendo usada
✅ Substituir imagem em todos os lugares de uma vez
✅ Deletar imagens não utilizadas

**Como acessar:**
1. Ao clicar em um campo de imagem
2. Clique em "Select" (não em Upload)
3. Você verá todas as imagens já enviadas

## Boas Práticas

### ✅ Fazer

- Otimizar imagens antes do upload
- Usar nomes descritivos (hero-turismo.jpg, logo-cliente-abc.png)
- Manter proporções corretas para cada seção
- Testar em dispositivos móveis
- Usar imagens de alta qualidade
- Adicionar texto alternativo (ALT) para acessibilidade

### ❌ Evitar

- Fazer upload de imagens gigantes (> 2MB)
- Usar imagens pixeladas ou de baixa qualidade
- Enviar imagens com proporção errada
- Esquecer de otimizar
- Usar stock photos genéricas demais
- Ignorar direitos autorais

## Direitos Autorais

<Callout type="error" title="Atenção Legal">
Use apenas imagens que você tem direito de usar:
- Fotos próprias da empresa
- Imagens compradas (shutterstock, etc.)
- Imagens gratuitas com licença (Unsplash, Pexels)
- Nunca use imagens do Google sem verificar licença!
</Callout>

## Sites de Imagens Gratuitas

Caso precise de imagens stock:

- **Unsplash**: unsplash.com (alta qualidade, grátis)
- **Pexels**: pexels.com (fotos e vídeos)
- **Pixabay**: pixabay.com (vasto catálogo)

Sempre verifique a licença antes de usar.

# Trabalhe Conosco

Página para atração de talentos.

## Acesso
Menu lateral: **Trabalhe Conosco Page**

## Conteúdo

### Textos Institucionais
*   Mensagem de boas-vindas aos candidatos.
*   Cultura da empresa.

### Benefícios
*   Atualize a lista de benefícios oferecidos (VR, VT, Saúde, etc.).

### Vagas
*   Se houver listagem de vagas, use **Add Item** para criar nova vaga.
*   Ou verifique se há um e-mail de contato para envio de currículos.

# Opções de Viagem

Página que lista as modalidades de serviços oferecidos.

## Acesso
Menu lateral: **Opções de Viagem Page**

## Categorias
Gerencie os blocos para cada tipo de viagem:
*   Aéreo
*   Hotelaria
*   Seguro Viagem
*   Incentivo
*   Etc.

## Edição
Para cada opção, você pode editar:
*   **Título**
*   **Descrição curta**
*   **Ícone/Imagem**

<Callout type="info">
  Mantenha as descrições concisas e focadas nos benefícios para o cliente.
</Callout>

# Página Eventos

Divulgue os serviços de eventos corporativos.

## Acesso
Menu lateral: **Eventos Page**

## Seções Editáveis

### Conteúdo Principal
*   **Título e Descrição:** Explique os tipos de eventos que a empresa organiza.
*   **Destaques:** Congressos, Feiras, Convenções, etc.

### Galeria
*   Adicione fotos de eventos realizados.
*   Use imagens de alta qualidade horizontal.

### Informações Adicionais
Se houver detalhes técnicos ou FAQ, verifique também se existe uma seção **Eventos Info Page** no menu.

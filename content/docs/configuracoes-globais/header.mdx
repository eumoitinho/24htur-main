# Menu (Header)

Menu superior presente em todas as páginas.

## Como editar

1. No Sanity Studio, vá em **Site Settings** > **Header Navigation**.
2. Gerencie os itens do menu e telefone.

## Componentes

### Menu Items
*   **Label:** Texto do link (ex: "Sobre").
*   **Link:** URL (ex: `/sobre`).
*   **Dropdown:** Marque para criar submenus (ex: Serviços > Corporativo).

### Phone Number
Telefone exibido no topo (com DDD).

<Callout type="warning" title="Links">
  Links internos devem começar com `/`. Links externos devem ter `https://`.
</Callout>
